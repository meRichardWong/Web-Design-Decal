There are two ways to implement the same functionality. One solution is shorter and more elegant than the other, but you can implement Dress Drizzy in whatever way makes sense to you.

The first solution is to show the corresponding outfit for every outfit selection. Pay attention to the ids in the HTML. Afterwards use ".fadeIn()" and ".fadeOut()" to add functionality to the "STRIP ME" button.

Setting Up Your .js File

Create a new js file in the assets/js folder. Call it "script.js". Then insert the following code in your js to set up your file:

$(document).ready(function(){
});

Now your js document is ready to run!

Method 1

Logic: When I click on a specific outfit-selection box, it will show the corresponding outfit and hide all the others. If I click "#s1", show "#o1", and hide "#o2", "#o3", "#o4", "#o5", and the "#body". Repeat the same for all outfits.

Let's start by setting up our click event. Hover below:

$("").click(function(){
});

What goes in the quotes? The element that you want to click on to make something happen (to make the function run). Let's click on "#s1" and show the corresponding outfit "#o1". Hover below:

$("#s1").click(function(){
$("#o1").show();
});

Now we have to hide all the other outfits and Drake's body! Hover below:

$("#s1").click(function(){
$("#o1").show();
$("#o2").hide();
$("#o3").hide();
$("#o4").hide();
$("#o5").hide();
$("#body").hide();
});

Repeat the same click event for all outfit-selections and outfits.

Method 2

Just like Method 1, let's set up our click event.

$("").click(function(){
});

What goes inside the quotes? When we click on any of the outfit-selection boxes, we want the other outfits and Drake's body to fadeOut. Hover below:

$(".outfit-selection").click(function(){
$(".outfit").fadeOut();
$("#body").fadeOut();
});

However, running this code seems to do nothing when we click on any of the outfit-selection boxes! This is because our function is not specific enough. Which outfit-selection will correspond to which outfit? We can accomplish this in jQuery using variables, "this", attr(), and slice(). Notice that there is a relationship between the ids for outfit-selections and outfits. When we click on "#s1", we want to only show outfit "#o1". How can we assign a value to a variable equal to the number in the id? Use "this", .attr() and .slice().

$(".outfit-selection").click(function(){
$(".outfit").fadeOut();
$("#body").fadeOut();
var num = $(this).attr("id").slice(1);
});

Using .attr() allows us to take the "id" of this specific outfit-selection that you click on. Using .slice() allows us to construct a new jQuery object containing a subset of the elements specified by the start position, 1, and also corresponds to this specific outfit-selection. Great! Now, we can use the integer value, num, to add to the "#o" part of the id for outfits.

$(".outfit-selection").click(function(){
$(".outfit").fadeOut();
$("#body").fadeOut();
var num = $(this).attr("id").slice(1);
var outfit = "#o" + num;
});

Lastly, we need the outfit to fadeIn! We can call certain outfits using the variable, outfit, inside the parentheses of $().fadeIn. Note that we don't need a "" inside the parentheses because we just defined the variable outfit with the id.

$(".outfit-selection").click(function(){
$(".outfit").fadeOut();
$("#body").fadeOut();
var num = $(this).attr("id").slice(1);
var outfit = "#o" + num;
$(outfit).fadeIn;
});

Activating the Strip Button

Logic: When we click on the strip-button, all outfits will fadeOut and Drake's body will fadeIn.

$("#strip-button").click(function(){
$(".outfit").fadeOut();
$("#body").fadeIn();
});
