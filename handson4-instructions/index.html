Changing the Font

This default font just doesn't cut it. Pick and import your own font from Google's expansive font library!

  1. Browse through the fonts or search for a font you like using the search bar or category checkboxes on the right side of the page. (We used Open Sans and Playfair Display.)
  2. Click on the red circular plus "+" icon in the upper right corner of the font.
  3. Click on the pop-up on the bottom of the page that says "_ Families Selected". Click on the "Customize" tab and select the font-weights you wish to use. You may select as many fonts and font-weights as you would like to use, but note that more fonts and font-weights make your page load slower. For the purpose of this handson, select two to three font-weights.
  4. Copy and paste the provided HTML code in the "Embed" tab into the head tag of your index.html and the provided CSS code into the appropriate selector of your CSS page (Hint: applying font-family to the body of your CSS file will make all of your text that font).

Now your font is properly linked to your HTML and is styled in your CSS! Yes, it's that easy. Thanks, Google.

How should I organize the divs in my webpage?
Since we need to have all elements centered (header, subheader, input box, button, social media icons), it is convenient to place them all in the same container. Then each element can have its own div and id, so that you can style them separately. How do I fill the entire window with my background image using CSS?
You'll need to add the following to your body:
body {
width: 100%; (to fill the entire width of the window)
margin: 0; (to make the image touch the edge of the window)
height: ___px; (defines height so that we are able to vertically center our text)
overflow: hidden; (fits image to window size so that text always appears centered)
}
And the following to your image in CSS:
image id or class {
width: 100%;
position: fixed;(don't worry about what this means just yet. More positioning to come next week!)
}
How do I center my text container horizontally?
container class or id {
display: block; (only block elements can use margin: 0 auto)
margin: 0 auto;
}
How do I center my text container vertically?
container class or id {
position: relative; (more about position to come next class!)
top: 50%; (pushes container down 50% of window height from top of page)
transform: translateY(-50%); (pushes container up 50% of container height from bottom of page)
}
How do I give my background image a dark overlay to make my light text more legible?
Add a div to your HTML with the id "background-overlay". Apply the following styles to your CSS:
#background-overlay {
background color: black; (puts a black block over the image)
opacity: 0.4; (makes overlay more transparent...try various opacities from 0.0-1.0)
height: 100%;
width: 100%;
position: fixed; (more about position later!)
}
